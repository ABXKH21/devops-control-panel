# DevOps Control Panel

Internal operations management system for DevOps teams.

## Modules
- Task Log
- Deployment Manager
- Calendar
- Dashboard

## Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: PostgreSQL
- Proxy: Nginx
- Containers: Docker + Docker Compose

## Environments
| Environment | Purpose |
|-------------|---------|
| QA | Auto-deploy on push |
| Release | Manual trigger |
| Production | Manual approval |
